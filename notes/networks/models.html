Network Models

Network models define how communication happens in a network. Two major models exist: OSI and TCP/IP.

OSI Model (7 Layers)

  • Physical
  • Data Link
  • Network
  • Transport
  • Session
  • Presentation
  • Application

TCP/IP Model (4 Layers)

  • Link
  • Internet
  • Transport
  • Application

Key Difference

OSI is a theoretical model, while TCP/IP is a practical implementation.
